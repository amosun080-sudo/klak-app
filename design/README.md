KLAK Design System
===================

Overview
--------
This folder contains the design system, tokens, motion guidelines, and example React Native mockups for the KLAK mobile app — a premium, AI-powered fintech product for Africa.

What you'll find
- `tokens/` — color, typography, spacing JSON tokens.
- `components.md` — component library and usage guidelines.
- `motion.md` — motion system and animation specs.
- `screens.md` — high-fidelity screen specifications and realistic sample data.
- `figma-structure.md` — recommended Figma frame & component structure for handoff.
- `mockups/` — runnable React Native mockup components demonstrating the visual system.

How to use
- Read `tokens/*` to apply consistent colors, typography and spacing.
- Use `components.md` and `motion.md` to implement UI components and animations.
- Open `mockups/*.tsx` in the app or Expo to preview visual patterns.
