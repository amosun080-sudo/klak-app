# Premium Fintech Illustrations

Tone
- Minimal abstract shapes, soft gradients, subtle metallic gold highlights.
- Avoid literal bank imagery; prefer modern, geometric metaphors for flow and growth.

Assets
- Hero svg sequences for onboarding (3 variants).
- Micro-illustrations for empty states (savings, budgets, transactions).

Style
- Use limited palette: emerald, electric blue, gold highlights on dark backgrounds.
- Add faint glass strokes and light reflections for depth.

Delivery
- Provide 3 sizes (small, medium, large) and Lottie files for animated sequences.

Glassmorphism Rules
- Frost layer opacity: 6–10% white tint on dark backgrounds.
- Blur: 18–24px in Figma, CSS backdrop-filter equivalent for web.
- Border: 1px with rgba(255,255,255,0.06).
- Elevation: multi-layer soft shadows using `shadows.soft` and `shadows.elevated` from tokens.
