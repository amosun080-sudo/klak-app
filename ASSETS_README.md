# Assets Setup

## Missing Assets (Fixed Temporarily)

The following assets were missing and have been temporarily removed from the configuration to allow the app to run:

### Required Assets for Production:

1. **App Icons:**
   - `./assets/adaptive-icon.png` (1024x1024) - Android adaptive icon foreground
   - Standard app icon will be generated by Expo

2. **Notification Icon:**
   - `./assets/notification-icon.png` (96x96) - Android notification icon
   - Should be a simple monochrome icon

### Quick Setup:

1. Create the assets directory if it doesn't exist:
   ```bash
   mkdir assets
   ```

2. Add your icon files to the `assets/` directory

3. Update `app.config.ts` to reference the icons:
   ```typescript
   android: {
     adaptiveIcon: {
       foregroundImage: './assets/adaptive-icon.png',
       backgroundColor: '#060E07',
     },
   },
   plugins: [
     [
       'expo-notifications',
       {
         icon: './assets/notification-icon.png',
         color: '#00D68F',
       },
     ],
   ],
   ```
